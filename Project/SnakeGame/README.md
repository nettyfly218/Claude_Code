# Snake Game - 霓虹贪吃蛇

一个使用原生 HTML5 Canvas 和 JavaScript 构建的经典贪吃蛇游戏，采用 **Neon Retro Arcade (霓虹复古街机)** 风格设计。

## 在线游玩

直接在浏览器中打开 `index.html` 即可开始游戏。

## 游戏控制

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → / WASD | 移动蛇 |
| SPACE | 暂停/继续 |
| R | 重新开始 |

## 速度调节

- 使用底部 **Speed 滑块** 可实时调节游戏速度
- 向左拖动 = 加速（Turbo 模式）
- 向右拖动 = 减速（休闲模式）
- 速度设置自动保存，下次打开自动恢复

## 设计特色

### 视觉风格
- **霓虹配色方案**: 品红、青色、黄色、亮绿大胆碰撞
- **CRT 扫描线效果**: 模拟 80 年代老式显示器质感
- **动态 3D 背景网格**: 透视动画网格营造空间感
- **悬浮粒子系统**: 多彩粒子在背景中漂浮流动

### 字体设计
- **Orbitron**: 科技感科幻字体，用于标题、分数、游戏内文字
- **Share Tech Mono**: 终端风格等宽字体，用于界面正文

### 交互特效
- **霓虹按钮**: 悬停时扫光动画 + 发光增强
- **分数面板**: 闪光动画效果
- **食物动画**: 霓虹脉冲呼吸效果
- **粒子爆炸**: 吃到食物时绽放粒子特效
- **速度指示器**: 实时显示当前游戏速度等级（Turbo/Fast/Normal/Slow）

## 技术实现

- **HTML5 Canvas**: 游戏画面渲染
- **原生 JavaScript**: 游戏逻辑，无第三方依赖
- **CSS3 动画**: 所有界面动效
- **localStorage**: 最高分和速度设置持久化存储
