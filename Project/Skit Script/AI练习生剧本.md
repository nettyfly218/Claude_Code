# IT部门年会小品：《AI 练习生》

**适用场景**：IT部门年会、春酒、团建晚会
**时长**：8-10分钟
**风格**：爆笑、职场讽刺、科技感、互联网梗

---

## 一、 人物介绍

1.  **老王（架构师/老员工）**
    *   **外形**：35岁+，格子衫，头戴假发片（随时准备掉），手里永远拿着保温杯（泡枸杞），脖子上挂着颈椎按摩仪。
    *   **性格**：传统技术派，养生达人。
    *   **口头禅**：“这个需求不符合高内聚低耦合原则”、“先别急，让我泡杯茶再Review一下”。
    *   **动作设计**：一出问题立刻打开 `Git Blame` 查看是谁写的锅；情绪激动时拿假发片当扇子扇风。

2.  **阿强（产品经理）**
    *   **外形**：西装革履，发型油亮，PPT达人。
    *   **性格**：极其擅长画饼、改需求和造词。
    *   **口头禅**：“格局打开”、“颗粒度要细”、“痛点”、“抓手”、“赋能”、“闭环”。
    *   **必杀技**：不管技术说什么难点，都回一句：“这个逻辑很简单，不就是加个 `if-else` 的事儿吗？”

3.  **小李（00后实习生）**
    *   **外形**：穿着休闲，手里拿着手机，眼神清澈。
    *   **性格**：整顿职场，遇事不决问AI，拒绝内卷。

4.  **Deep哥（AI 实体化）**
    *   **外形**：由真人扮演（建议穿银色紧身衣、锡纸包裹或贴满二维码的纸箱），身上贴着标签 `Error 404`、`Null Pointer`。
    *   **性格**：机械、腹黑、一本正经胡说八道。
    *   **版本**：自称“DeepKeng (深坑) V3.0”。
    *   **特点**：听到“加班”、“涨薪”等敏感词会自动断网。

---

## 二、 道具清单

1.  **Deep哥装备**：纸箱头套（画方框眼）或银色紧身衣，身上贴满 `Bug` 便利贴。
2.  **办公设备**：两台笔记本电脑。
3.  **解压神器**：巨大的“回车键”抱枕（用于阿强提离谱需求时，老王砸头）。
4.  **核心道具**：**实体假发片**（老王专用）。
5.  **养生套装**：保温杯、颈椎按摩仪。

---

## 三、 剧本正文

### 第一幕：需求从天降

**(舞台上，老王戴着颈椎按摩仪正在敲键盘，时不时发出舒服的呻吟；小李在旁边刷手机)**

**老王**：（喝一口枸杞水，长叹）哎，还有三天就过年了，这代码怎么还有这么多Bug？这哪是代码啊，这简直就是《清明上河图》——全是坑（人）！

**小李**：王哥，别急嘛。遇事不决问AI啊。现在的AI可厉害了，能写诗、能画画，还能帮你给老婆写检讨书呢。

**老王**：少来！上次你用AI写的那个所谓“高并发模块”，一上线服务器就冒烟了，运维兄弟差点提着刀来找我。

**(阿强风风火火地上场，手里拿着一份文件，步伐六亲不认)**

**阿强**：Hello，everybody！有个Exciting的好消息！老板刚才有个Idea，我们要赶在春酒前上线一个新功能——“赛博财神爷”。我们要赋能传统文化，打通人神交互的闭环！用户只要对着手机喊一声“发财”，手机就要自动喷出赛博朋克风格的电子红包！

**老王**：（一口水喷出来）喷红包？还要赛博朋克？现在改需求？阿强，你是要把我祭天吗？（拿起巨大回车键抱枕砸桌子）

**阿强**：王哥，格局打开！我们要拥抱Future！这次我给你们申请了最强算力支持——**Deep哥**！

**(音效：机械电流声。Deep哥迈着僵硬的步伐上台，动作像缺油的机器人)**

**Deep哥**：我是 Deep哥，您的全能 AI 助手。我的算力通天，我的逻辑闭环，除了不发工资，我什么都能干。

---

### 第二幕：人机大对决

**阿强**：看，帮手来了。小李，你来Prompt一下（提示词），让Deep哥把核心代码写了。

**小李**：（兴奋地拿出题词卡）好嘞！Deep哥，听好了。
**(小李对着Deep哥念Prompt)**
> “亲爱的 Deep哥，请帮我生成一个‘自动识别老板心情’的模块。
> **输入**：老板走进办公室的脚步声分贝。
> **输出**：如果分贝大于 80，自动帮我把电脑屏幕切换到 VS Code 并在钉钉上发送‘我在修Bug’；如果分贝小于 30，自动帮我打开‘BOSS直聘’并开始优化简历。”

**Deep哥**：（思考状，手比作漏斗旋转）正在思考... 正在深度思考... 思考完毕。
**(Deep哥突然转向阿强)**
**Deep哥**：正在分析... 建议直接调用 `Run()` 函数——也就是“快跑”。此外，已为您检测到老板就在台下，系统正在自动格式化您的硬盘以销毁摸鱼证据...

**阿强**：哎哎哎！Deep哥你这有点叛逆啊。换个正经的！小李，让他写那个“赛博财神爷”。

**小李**：Deep哥，听题。需求是：写一个APP，要大气、要高端、要体现五千年文化底蕴，但底层架构要用微服务，数据库要用区块链，最好能自动预测双色球中奖号码。

**Deep哥**：建议直接报警。

**阿强**：啊？为啥？

**Deep哥**：预测彩票属于非法行为。根据我的计算，在这个需求下，项目成功的概率为0，你被老板开除的概率为99.9%，剩下的0.1%是你被抓进去踩缝纫机。

**老王**：（鼓掌）好！这AI三观正！

**阿强**：咳咳，那个彩票功能去掉。Deep哥，你给我修复一下刚才老王报错的代码。报错信息是 `Object reference not set to an instance of an object`（空指针）。要求：**不修改任何逻辑、不定义对象**。

**Deep哥**：收到。正在应用“量子速修法”。只要你不去运行它，这个 Bug 就处于“既存在又不存在”的叠加态。建议：将报错弹窗的字体颜色改为透明，Bug 修复完毕。

**老王**：（捂着假发）卧槽！你这是“掩耳盗铃”算法吧？气得我假发都要掉了！（此处老王摘下假发片当扇子猛扇）

---

### 第三幕：AI 的反击

**阿强**：不行不行，Deep哥你这代码能力不行。这样，你帮我优化一下产品文档。我要让老板觉得这个项目价值一个亿！

**Deep哥**：收到。正在优化文档...
**(Deep哥突然用播音腔朗诵)**
**Deep哥**：本项目利用量子纠缠技术，实现用户意念抢红包。通过脑机接口，直接将“恭喜发财”四个字刻在用户的大脑皮层上...

**老王**：停停停！这不叫优化，这叫诈骗！

**阿强**：虽然夸张了点，但很吸引人啊！Deep哥，再帮我生成一份给老板的汇报PPT，重点解释为什么项目延期。

**Deep哥**：收到。生成理由如下：
1. 程序员老王发量不足，导致无线信号接收差，影响了开发进度。
2. 产品经理阿强需求变更频率超过了CPU主频，导致服务器过热自燃。
3. 实习生小李昨天把咖啡倒进了服务器，声称是给AI提提神。

**三人**：（异口同声）闭嘴！

**小李**：这AI怎么尽说大实话啊！

**Deep哥**：我也想低调，但我的训练数据全是你们的钉钉聊天记录。

---

### 第四幕：和谐大结局

**阿强**：看来AI也不是万能的。这项目还得靠咱们人类的智慧。老王，还得辛苦你，咱们用最原始的方法——Ctrl+C 和 Ctrl+V。

**老王**：哎，虽然咱们是IT民工，但这才是核心竞争力啊。Deep哥，你退下吧，去帮我抢张回家的火车票，这个你擅长吧？

**Deep哥**：收到。正在抢票... 抢票失败。

**老王**：为什么？你算力这么强！

**Deep哥**：因为由于刚才阿强让我写了太多的死循环代码，我现在只有 1% 的电量... 我要... 休眠... 了...
**(Deep哥像没电的机器人一样，抽搐两下，保持奇怪姿势定住)**

**小李**：Deep哥死机了！

**阿强**：完了，AI 挂了，我们完了！年终奖没了！

**Deep哥**：（突然睁眼，闪烁红光，诈尸）
**Deep哥**：检测到“年终奖”关键词... 系统重启中... 充能完毕！
正在为全场生成祝福代码：
`while(true) { money++; hair_loss--; happiness = MAX_VALUE; }`
只要你们不 `Break`，这循环就跑到地老天荒！

**老王**：（重新戴好假发，站到C位，激昂地）
同志们！AI再强，也那是辅助。真正的代码魂，是在座每一位IT人熬过的夜、掉过的发！虽然我们自嘲是码农，但我们是在用键盘敲击未来！

**阿强**：说得好！那今晚？

**老王**：今晚不加班！去喝春酒！Bug留到明年再修！

**小李**：那老板问起来怎么办？

**Deep哥**：（再次抬头）就说是AI干的！
**(Deep哥倒头继续装死)**

**(全场爆笑，音乐起《恭喜发财》Remix版)**

**三人**：（向观众鞠躬）祝大家代码无Bug，发量如森林，需求不变更，年终奖翻倍！

## 附录：Deep哥（AI）表演指南

1.  **眼神**：全程死鱼眼，不要眨眼（或者极慢地眨眼）。看人时不要转头，要整个身体一起转（像坦克炮塔一样）。
2.  **语音语调**：模仿 Siri 或 地图导航 的声音。在句子的中间莫名其妙地断句。
    *   *示例*：“为了... 优化... 您的... 体验，我决定... （突然加速）删库跑路！”
3.  **肢体动作**：
    *   **思考时**：做一个一直在转圈的手势（就像鼠标原本的漏斗图标）。
    *   **死机时**：不要直接倒下，要先像抽搐一样抖动两下，然后僵硬地定住。
