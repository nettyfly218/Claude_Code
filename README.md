# Claude Code 学习笔记

本仓库用于记录 Claude Code AI 助手的学习笔记和技能配置。

## 项目结构

```
Claude_Code/
├── .claude/          # Claude Code 配置目录
├── Agents/           # Agent 配置文件列表
├── MCP/              # MCP 服务器配置列表
├── Memory/           # 记忆数据库文件
├── Prompt/           # Prompt 技能文档
├── Skills/           # Claude Skills 技能模块
├── 学习笔记/          # 个人学习笔记
└── 背单词/            # 词汇学习资料
```

## 主要内容

### Skills（技能模块）
包含多种 Claude Code 技能：
- `algorithmic-art` - 算法艺术生成
- `brand-guidelines` - 品牌指南应用
- `canvas-design` - Canvas 设计工具
- `docx` - Word 文档处理
- `frontend-design` - 前端界面设计
- `pdf` - PDF 文档处理
- `pptx` - PowerPoint 演示文稿
- `web-artifacts-builder` - Web 构建工具
- `xlsx` - Excel 电子表格
- 以及更多...

### Prompt
包含 Claude Code 的技能说明和使用指南。

### Agents
记录可用的 Agent 列表及其功能。

### MCP
记录配置的 MCP（Model Context Protocol）服务器。
